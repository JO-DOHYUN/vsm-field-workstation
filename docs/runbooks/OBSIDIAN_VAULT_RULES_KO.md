# OBSIDIAN_VAULT_RULES_KO

이 프로젝트 루트를 Obsidian vault로 열 때의 문서 규칙이다.

## Link Rules
- 핵심 md는 가능한 Obsidian 내부 링크 형식을 둔다.
- 루트 hub는 [[../../INDEX]], docs hub는 [[../README]], history hub는 [[../../history/INDEX]]다.
- 코드 파일 경로는 링크보다 literal path로 적어도 된다.

## Noise Rules
아래는 vault 탐색에서 artifact/noise로 본다.
- `.vs/`
- `out/`
- build/deploy 산출물
- 대용량 zip
- screenshot png
- temporary capture/log output

## History Rules
현재 판단에 필요한 사실은 [[../../BRIEF]]에 남기고, 오래된 시행착오와 실패 기록은 [[../../history/INDEX]] 아래로 보낸다.
